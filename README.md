# AUKUS-Trade-PolicyBrief
This project analyzes whether the 2021 AUKUS security agreement between Australia, the UK, and the US led to trade diversion from China or trade creation with AUKUS partners. It includes a policy brief, replication code in R (.Rmd, .html), and a gravity model estimated using PPML. Data sources include UN Comtrade, World Bank, CEPII, and GTA.
